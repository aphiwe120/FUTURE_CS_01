# Altoro Mutual: Vulnerability Assessment Report
## Project Overview
This project involved a comprehensive security assessment of the Altoro Mutual (demo.testfire.net) banking application. The goal was to identify potential attack vectors, assess risk levels, and provide actionable remediation strategies for the client.

## Scope

Target: http://demo.testfire.net

Environment: Kali Linux

Methodology: Passive Reconnaissance, Automated Vulnerability Scanning, and Manual Header Analysis.

## Tools Used

Nmap: For network reconnaissance and service discovery.

OWASP ZAP: For automated web application security testing and header auditing.

Canva: For professional executive reporting and dashboard design.

## Key Findings

1 High Risk: Insecure plaintext communication (Port 80/HTTP).

3 Medium Risks: Missing security headers (CSP, X-Frame-Options, X-Content-Type-Options).

Multiple Low Risks: Information Disclosure (Server versioning) and insecure cookie attributes.
